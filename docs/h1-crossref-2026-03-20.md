## Key Architectural Differentiators

The following design decisions eliminate entire Ubiquiti attack chains:

1. **No XFF trust** → XFF spoofing chain (HIGH-01 + HIGH-02) impossible
2. **No UDP discovery** → topology leak + amplification (MEDIUM) impossible
3. **Hardware-bound DB keys** → credential theft requires physical access
4. **E2EE envelope** → session hijacking requires both token AND envelope key
5. **MGMT-only Inform** → phantom injection limited to MGMT VLAN (not Guest/LAN)
6. **3-stage adoption** → phantom devices can't extract authkeys (SSH verify + HW fingerprint required)
7. **No MongoDB/RabbitMQ/Erlang** → entire service stack attack surface eliminated
